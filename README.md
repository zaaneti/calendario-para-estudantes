# 📅 Student Study Calendar

A simple and intuitive web-based calendar designed to help students organize their study routine, deadlines, exams, and academic tasks.

## 🚀 Live Demo

Access the project here:
https://zaaneti.github.io/calendario-para-estudantes/

---

## 🎯 Purpose

This project was created to help students:

- Organize study schedules
- Track important deadlines
- Improve time management
- Maintain academic productivity
